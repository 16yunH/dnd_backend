# 检定规则

## d20 检定

D&D 5e 的核心机制是 d20 检定：掷一枚二十面骰（d20），加上相关调整值，与目标数值（DC 或 AC）比较。

**公式**：d20 + 调整值 + 熟练加值（若熟练） vs 目标数值

- 结果 ≥ 目标 = 成功
- 结果 < 目标 = 失败
- d20 掷出 20 = 大成功（自动成功，攻击时为重击）
- d20 掷出 1 = 大失败（自动失败）

## 优势与劣势

- **优势**：掷两枚 d20，取较高值
- **劣势**：掷两枚 d20，取较低值
- 优势与劣势互相抵消；多个优势/劣势不叠加，只算一次

## 熟练加值

所有角色共享熟练加值，随等级提升：

| 角色等级 | 熟练加值 |
|----------|----------|
| 1-4 | +2 |
| 5-8 | +3 |
| 9-12 | +4 |
| 13-16 | +5 |
| 17-20 | +6 |

## 属性检定

属性检定是衡量角色尝试某项行动是否成功的通用检定。

**公式**：d20 + 属性调整值 + 熟练加值（若该技能熟练）

### 属性检定类型

- **力量检定**：举重、推门、挣脱束缚、游泳
- **敏捷检定**：保持平衡、表演杂技、灵巧操作
- **体质检定**：忍耐极端环境、持续奔跑、抵抗疾病
- **智力检定**：回忆知识、解读文字、推理分析
- **感知检定**：察觉隐藏事物、读心、追踪猎物
- **魅力检定**：说服他人、表演、威吓

## 技能检定

技能检定是属性检定的子类，对应特定领域。若角色拥有某技能的熟练项，则在相关属性检定上额外加入熟练加值。

### 技能与属性对应

- 力量：运动
- 敏捷：体操、巧手、隐匿
- 智力：奥秘、历史、调查、自然、宗教
- 感知：驯兽、洞悉、医药、察觉、求生
- 魅力：欺瞒、威吓、表演、游说

## 对抗检定

当两个角色互相竞争时，进行对抗检定：双方各掷 d20 + 调整值，结果高者胜出。平局时通常维持现状，或由 DM 裁定。

## 被动属性值

被动属性值 = 10 + 属性调整值 + 熟练加值（若熟练）。代表角色在未主动尝试时的默认感知水平。最常见的被动属性值是被动察觉。

## 群体检定

当一组人同时进行同一检定时，每人都掷骰；若半数以上成功，则整组成功。
