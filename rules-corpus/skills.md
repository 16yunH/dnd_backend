# 技能

D&D 5e 共有 18 项技能，每项技能关联一个属性。进行技能检定时掷 d20 + 属性调整值 + 熟练加值（若熟练）。

| 技能 | 英文ID | 关联属性 |
|------|--------|----------|
| 运动 | athletics | 力量 |
| 体操 | acrobatics | 敏捷 |
| 巧手 | sleight_of_hand | 敏捷 |
| 隐匿 | stealth | 敏捷 |
| 奥秘 | arcana | 智力 |
| 历史 | history | 智力 |
| 调查 | investigation | 智力 |
| 自然 | nature | 智力 |
| 宗教 | religion | 智力 |
| 驯兽 | animal_handling | 感知 |
| 洞悉 | insight | 感知 |
| 医药 | medicine | 感知 |
| 察觉 | perception | 感知 |
| 求生 | survival | 感知 |
| 欺瞒 | deception | 魅力 |
| 威吓 | intimidation | 魅力 |
| 表演 | performance | 魅力 |
| 游说 | persuasion | 魅力 |
